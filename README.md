# 🚀 Gravity Shaft

## 📱 Game Overview

**Gravity Shaft** is a 2D mobile endless runner designed for vertical (portrait) gameplay.

The player controls a character moving upward through a narrow neon corridor. By tapping the screen, the player flips gravity and switches between the left and right walls to avoid obstacles.

The goal is to survive as long as possible while navigating increasingly difficult patterns.

---

## 🧠 Core Idea

Instead of jumping, the player **switches sides**:

* Left wall ⬅️
* Tap → Right wall ➡️

This creates a fast-paced, one-touch gameplay experience идеально подходящий для мобильных устройств.

---

## 🎯 Controls

| Input       | Action                               |
| ----------- | ------------------------------------ |
| Tap / Space | Flip side (change gravity direction) |
| R           | Restart (for testing in editor)      |

---

## ⚙️ Core Mechanics

* Automatic upward movement
* Gravity flip between left and right walls
* Obstacles spawn in the corridor
* Collision triggers Game Over
* Endless gameplay loop
* Score increases over time

---

## 🕹️ Gameplay Loop

1. Player starts moving upward automatically
2. Obstacles begin spawning
3. Player taps to switch sides
4. Collision → Game Over
5. Restart and try again

---

## 🚀 MVP (Minimum Viable Product)

The first playable version includes:

* Player movement (auto upward)
* Side switching mechanic (gravity flip)
* Basic obstacles (at least one type)
* Obstacle spawning system
* Collision detection
* Game Over state
* Restart functionality
* Score system (based on survival time)

---

## ✅ Definition of Done

The prototype is considered complete when:

* The game runs without crashes
* Player movement and side switching work correctly
* Obstacles spawn and move consistently
* Collisions correctly trigger Game Over
* Restart works properly
* Score updates over time
* Full gameplay loop is functional
